## 📚 模块概览

本模块介绍**检索增强生成** (RAG) 技术，通过将外部知识库与生成式模型结合来改进输出质量。

### 核心理念

- **动态知识检索**：在生成时从外部源检索相关信息
- **知识增强**：用检索到的文档增强模型上下文
- **事实性保证**：通过检索提高答案的准确性
- **知识库管理**：维护和更新外部知识库

## 📁 主要内容

- RAG架构设计
- 检索算法与优化
- 知识库构建与维护
- 检索器与生成器协调
