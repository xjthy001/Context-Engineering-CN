# 心智模型

本目录包含用于理解NOCODE语境工程的直观心智模型。

## 可用模型

1. **花园模型** (01_garden_model.md) - 将语境管理想象为园艺
2. **预算模型** (02_budget_model.md) - 将tokens视为财务资源
3. **河流模型** (03_river_model.md) - 语境作为流动的河流
4. **生物心理社会模型** (04_biopsychosocial_model.md) - 整体性视角
5. **炼金术模型** (05_alchemy_model.md) - 转化性语境工程

每个模型提供理解和应用语境工程原则的不同视角。

详见各文件。
