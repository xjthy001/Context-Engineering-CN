# 炼金术模型:转化心智模型

## 简介

炼金术模型从古老的炼金术实践中汲取灵感,古代炼金术士寻求将基础材料转化为黄金。在上下文工程中,我们转化原始输入(基础材料)为精致输出(黄金)。这一旅程涉及特定阶段、过程和转化,每一步都使材料更接近其理想形式。

**核心隐喻**: 上下文工程就像炼金术——通过精心的过程和转化将原始输入转化为精致输出。

## 伟大工作(Magnum Opus)

在炼金术中,"伟大工作"是从基础材料创造哲人石或黄金的过程。在上下文工程中,它是将简单输入转化为宝贵洞察的旅程。

```
炼金转化过程

原初物质(Prima Materia)          哲人石
    ↓                                  ↑
[原始输入]                        [精致输出]
    ↓                                  ↑
    └──→ 转化阶段 ──→ 精炼 ──→ 转化 ──┘
         (Nigredo)    (Albedo)  (Citrinitas) (Rubedo)
```

## 四个阶段

### 阶段1: Nigredo(黑化) - 分解

原初物质的黑化和分解。

**在上下文工程中**: 问题分解,初始分析,去构造。

```
Nigredo过程:
输入材料
    ↓
[分解]
├── 识别组件
├── 分离关注点
├── 暴露结构
└── 解构复杂性
    ↓
分离的元素
```

**技术**:

```markdown
## Nigredo提示模板

让我们分解这个___:

1. 核心组件是什么?
2. 关键关系是什么?
3. 隐藏的假设是什么?
4. 基本元素是什么?
5. 潜在问题是什么?

首先分离,然后我们将重建。
```

**标志**:
- 混淆先于清晰
- 解构先于重建
- 分离先于整合
- 问题先于答案

**Nigredo问题**:
```markdown
问题类型: 分析性、探索性
示例:
- "问题的组成部分是什么?"
- "这个系统如何分解?"
- "核心元素是什么?"
- "潜在挑战是什么?"
```

### 阶段2: Albedo(白化) - 净化

洗涤和净化材料。

**在上下文工程中**: 澄清,精炼,移除不必要的内容。

```
Albedo过程:
分离的元素
    ↓
[净化]
├── 移除噪音
├── 澄清意义
├── 精炼理解
└── 提取精华
    ↓
纯粹元素
```

**技术**:

```markdown
## Albedo提示模板

现在让我们净化我们的理解:

1. 什么是核心真理?
2. 什么是干扰?
3. 我们如何澄清混淆?
4. 什么是本质?
5. 最简单的表达是什么?

剥离不必要的,揭示纯净的。
```

**标志**:
- 清晰浮现
- 简单从复杂中出现
- 模式变得可见
- 理解深化

**Albedo问题**:
```markdown
问题类型: 澄清性、精炼性
示例:
- "最简单的解释是什么?"
- "核心概念是什么?"
- "我们如何澄清这一点?"
- "真正重要的是什么?"
```

### 阶段3: Citrinitas(黄化) - 启蒙

太阳的黄光带来启蒙。

**在上下文工程中**: 洞察,综合,连接,理解。

```
Citrinitas过程:
纯粹元素
    ↓
[启蒙]
├── 连接模式
├── 综合洞察
├── 整合理解
└── 实现潜力
    ↓
启蒙理解
```

**技术**:

```markdown
## Citrinitas提示模板

现在让我们综合洞察:

1. 模式如何连接?
2. 出现了什么更深层次的真理?
3. 我们能得出什么洞察?
4. 这如何一起工作?
5. 更大的图景是什么?

从碎片中,创造理解。
```

**标志**:
- 洞察出现
- 连接形成
- 意义具体化
- 理解综合

**Citrinitas问题**:
```markdown
问题类型: 综合性、整合性
示例:
- "这些碎片如何一起适配?"
- "出现了什么模式?"
- "关键洞察是什么?"
- "这意味着什么?"
```

### 阶段4: Rubedo(红化) - 完成

材料的最终完善和转化。

**在上下文工程中**: 实施,完善,完成,交付。

```
Rubedo过程:
启蒙理解
    ↓
[完善]
├── 实施解决方案
├── 完善细节
├── 优化结果
└── 完成工作
    ↓
黄金(完善输出)
```

```markdown
## Rubedo提示模板

1. 可行的步骤是什么?
2. 我们如何实施这一点?
3. 需要什么细节?
4. 我们如何优化?
5. 最终形式是什么?

从理解到实现。
```

**标志**:
- 实施清晰
- 细节完善
- 可行性确认
- 完成达成

**Rubedo问题**:
```markdown
问题类型: 实施性、完善性
示例:
- "我们如何构建这个?"
- "具体步骤是什么?"
- "我们如何完善这一点?"
- "最终交付是什么?"
```

## 炼金过程

### 过程1: 溶解(Solutio)

溶解固定结构为流动状态。

```markdown
## 溶解技术

将___从固定形式转化为流动理解:

- 溶解硬性假设
- 软化刚性思维
- 流动到可能性中
- 探索流体选项

示例:
"让我们溶解我们关于___的假设,
探索可能性空间。"
```

### 过程2: 凝结(Coagulatio)

将流体元素凝结为固定形式。

```markdown
## 凝结技术

从流动可能性中结晶清晰形式:

- 从选项中固化决策
- 从理论中具体化实践
- 从抽象中形成结构
- 从流体中创造固定

示例:
"从这些可能性中,让我们凝结
一个具体的方法。"
```

### 过程3: 分离(Separatio)

将元素分离为成分。

```markdown
## 分离技术

将复杂系统分离为组件:

- 分开关注点
- 隔离变量
- 区分方面
- 提取元素

示例:
"让我们分离___的层次
以独立理解每一层。"
```

### 过程4: 结合(Coniunctio)

将分离的元素结合成新的整体。

```markdown
## 结合技术

合并元素成综合整体:

- 集成组件
- 合并视角
- 统一元素
- 创建综合

示例:
"现在让我们结合这些元素
成一个统一的方法。"
```

### 过程5: 发酵(Fermentatio)

通过发酵促进有机转化。

```markdown
## 发酵技术

允许有机发展和成熟:

- 让想法发酵
- 允许自然演变
- 培育发展
- 耐心转化

示例:
"让这些概念发酵,让我们
看看什么有机地浮现。"
```

### 过程6: 蒸馏(Distillatio)

通过蒸馏提取精华。

```markdown
## 蒸馏技术

提炼到最纯净的形式:

- 蒸馏核心真理
- 提取精华
- 净化到本质
- 提炼到基础

示例:
"让我们蒸馏这个到其核心精华,
移除所有非必要的。"
```

### 过程7: 钙化(Calcinatio)

通过火烧掉杂质。

```markdown
## 钙化技术

通过强烈审查烧掉错误:

- 通过批判净化
- 通过火测试
- 烧掉弱点
- 在压力下强化

示例:
"让我们钙化这个方法——
对它施加批判性火焰。"
```

```

  ↓

  ↓

```

"通过整合两者,我们达到C..."
```

```

**实施**:

"最终完善到生产质量"

## 大师炼金术士心态

像大师炼金术士一样思考:

1. **耐心转化**: 急不得伟大工作
2. **精确测量**: 小心权衡每个元素
3. **过程信任**: 遵循自然阶段
4. **敏锐观察**: 注意转化的迹象
5. **实验精神**: 尝试不同过程
6. **尊重材料**: 理解输入的本质
7. **掌握火焰**: 控制转化强度
8. **追求完美**: 追求黄金标准

## 炼金格言

> "如上,如下;如内,如外" - 在所有层面上应用原则

> "急躁炼金术士只得铅" - 急不得转化

> "哲人石在旅程中,而非终点" - 过程即价值

> "分离才能纯化,纯化才能统一" - 通过阶段工作

> "第一物质包含黄金——你必须释放它" - 潜力已存在
