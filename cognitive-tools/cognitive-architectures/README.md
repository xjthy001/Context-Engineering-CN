# 认知架构

本目录包含组合多个认知工具的完整系统级架构。

## 架构目录

- **solver-architecture.md**: 问题解决系统架构
- **tutor-architecture.md**: 教育和辅导系统架构
- **research-architecture.md**: 研究助手系统架构
- **reconstruction-memory-architecture.md**: 记忆重建架构
- **field-architecture.md**: 场论推理架构
- **quantum-architecture.md**: 量子语义架构
- **unified_architecture.md**: 统一场架构
- **interpretability-architecture.md**: 可解释性架构

这些架构展示了如何构建完整的认知系统。
